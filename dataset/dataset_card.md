# Dataset Card

## Dataset Name

Adaption Legal Clause Simplification

---

## Overview

This dataset was created to fine-tune a Large Language Model for legal document simplification. Each record contains a legal clause paired with a plain-English version that preserves the original legal meaning while improving readability.

---

## Dataset Information

| Property      | Value                     |
| ------------- | ------------------------- |
| Domain        | Legal NLP                 |
| Language      | English                   |
| Task          | Legal Text Simplification |
| Format        | Input → Output            |
| Total Samples | 3,477                     |
| License       | MIT                       |

---

## Source

The dataset was developed using publicly available legal documents, including:

* CUAD (Contract Understanding Atticus Dataset)
* Privacy Policies
* Terms of Service
* Employment Agreements
* Lease Agreements
* Non-Disclosure Agreements (NDAs)
* Consumer Agreements

---

## Processing

The dataset was improved using the Adaption AutoScientist platform.

Processing steps included:

* Duplicate removal
* Data cleaning
* Formatting standardization
* Quality enhancement
* Plain-English rewriting

---

## Intended Use

This dataset is intended for:

* Legal document simplification
* Large Language Model fine-tuning
* NLP research
* Educational purposes
